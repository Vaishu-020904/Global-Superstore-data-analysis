# Global-Superstore-Data-Analysis

## 📊 Project 1: Exploratory Data Analysis (EDA) – Global Superstore Dataset  

An end-to-end **Exploratory Data Analysis** project using Python to uncover sales, profit, and customer insights.  

### 🚀 Features  
- Cleaned and prepared **50k+ rows of raw sales data** (handled missing values, duplicates, and datetime formatting).  
- Engineered new features like **unit price, order month/year** for deeper insights.  
- Visualized **5+ KPIs** including sales, profit, discounts, and shipping performance.  
- Identified **loss-making transactions** and analyzed profitability trends by segment and region.  
- Time-series analysis of **monthly sales & profit trends**.  

### 🛠️ Tools Used  
- **Python**: Pandas, NumPy, Seaborn, Matplotlib  

### 🔎 Key Insights  
- Discounts often increase sales but reduce profitability.  
- Customer Segments and Regions show different revenue and profit growth patterns.  
- Negative profit transactions highlight risky discounts and product categories.

## 📊 Project 2: Sales Analysis Dashboard – Excel & Power BI  

An **interactive dashboard** built using Excel (pivot tables, pivot charts) and Power BI, providing actionable insights on sales, profit, and customer performance.  

### 🚀 Features  

- Excel-based **pivot tables & pivot charts** for quick insights  
- Interactive **Power BI dashboard** with slicers and filters  
- Sales, profit, and margin analysis  
- Profit vs. loss separation  
- Customer and product performance breakdown  

### 📂 Project Structure  

- **Data/** → Raw sales dataset  
- **Excel/** → Excel workbook with pivot tables & pivot charts  
- **Dashboard/** → Power BI report file (.pbix)  

### 🔎 Key Highlights of the Dashboard:  

- ✅ $12.64M **Total Revenue** across all regions  
- ✅ $1.47M **Total Profit** with 51.2K Orders  
- ✅ **Top 10 Products** contributing the highest revenue  
- ✅ **Monthly Revenue & Profit Trends** to identify seasonality  
- ✅ **Regional & Country-wise analysis** with map visualization  
- ✅ **Sub-Category Profitability** to spot strong vs weak performers  

### 🛠️ Tools Used  

- **Excel** – Data cleaning, pivot tables, pivot charts  
- **Power BI** – Data modeling, interactive dashboards
